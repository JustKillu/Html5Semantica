HTML5: Semántica

En este ejemplo, veremos cómo utilizar la semántica HTML para mejorar la estructura y el significado de nuestras páginas web.

¿Qué es la semántica HTML?

La semántica HTML es el uso de las etiquetas HTML para reforzar el significado de la información en las páginas web.

En HTML5, se introdujeron nuevos elementos semánticos que permiten describir con mayor precisión el contenido de una página web.

  • header: Cabecera de la página. Es el elemento principal de la página y suele contener el logotipo, el título y el menú de navegación.
  • main: Cuerpo principal de la página. Es el elemento que contiene el contenido principal de la página, como el texto, las imágenes, los videos y los formularios.
  • article: Artículo o pieza de contenido independiente. Este elemento define un artículo o pieza de contenido independiente. Se utiliza para organizar el contenido de la página en secciones más pequeñas.
  • section: Sección de contenido. Este elemento define una sección de contenido. Se utiliza para organizar el contenido de la página en secciones más grandes.
  • aside: Contenido secundario o relacionado. Este elemento define contenido secundario o relacionado. Se utiliza para mostrar información adicional que no es esencial para el contenido principal de la página.
  • footer: Pie de página de la página. Este elemento define el pie de página de la página web. Es el elemento secundario de la página y suele contener información de copyright, información de contacto y enlaces a páginas relacionadas.
